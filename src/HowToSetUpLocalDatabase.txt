1. Download and install mysql on local machine, I use mysql 5.6.21
   username: root
   password: 1234
   port: 3306
   ** same as the default database attributes in aauction\settings.py
2. Pull code from staging branch. 
   //我看了一下，原来那个netmag里面所有的naming都是按照那个tutorial来的，但是里面的内容可以供我们参考。

3. run command in shell:
	#create models
	$ python manage.py makemigrations signups
	$ python manage.py makemigrations auctions
	
	#create tables
	$ python manage.py sqlmigrate signups 0001
	$ python manage.py sqlmigrate auctions 0001

	#see https://docs.djangoproject.com/en/1.8/intro/tutorial01/#activating-models for more info
